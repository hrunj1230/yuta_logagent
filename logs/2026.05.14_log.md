## 학습 내용

### 리스트(List)
**개념**
- 데이터를 순차적으로 저장하여 가독성을 높이고 편리하게 관리하기 위한 자료구조
- 순서가 있으며, 유동적으로 크기를 변경할 수 있는 순차적 형태의 자료구조

**기본 사용법**
```python
list = [1,2,3,4]
```

**리스트 조회**
```python
print(f"첫번째 list 는 {list[0]} 이다")
```

**리스트 요소 추가**
```python
list.append(5)  # 마지막에 요소 추가
```

**리스트 요소 변경**
```python
list[4] = 6  # 인덱스 4번 요소를 6으로 변경
```

**리스트 요소 삭제**
```python
del list[3]     # 인덱스 3번 요소 삭제
list.pop()      # 마지막 요소 삭제
list.pop(0)     # 첫번째 요소 삭제
```

### 함수(Function)
**개념 정리**
- 수학적 정의: 두 개의 변수 x, y에서, x가 일정한 범위 내에서 값이 변하고 y의 값이 정해질 때, x에 대하여 y를 이르는 말
- 정보통신 정의: 재활용하기 위해 묶어놓은 코드들의 집합

**함수 정의 예시**
```python
def make_bread(a, b):
    print("빵빵")
    return "빵"
```
